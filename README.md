# Template Toolkit for VS Code

Support for [Perl Template Toolkit](http://www.template-toolkit.org/index.html) syntax and snippets in Visual Studio Code.
## Features
File extensions:
-.tt
-.tpl

### Syntaxes:
SPECIAL for SHM (https://myshm.ru, https://docs.myshm.ru)
- TT2 {{ ... }}

### Snippets:
- BLOCK
- CALL
- DEFAULT
- END
- FOREACH
- GET
- IF ELSE
- IF ELSIF ELSE
- IF ELSIF
- IF
- INCLUDE
- INSERT
- MACRO
- PROCESS
- SET
- SWITCH
- USE
- WRAPPER
